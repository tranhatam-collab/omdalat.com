What is OMDALAT

OMDALAT is the Da Lat city node of OMDALA for work, hosting, community, trust, and real local value. It is not a tourism site, not a generic directory, and not the global runtime itself. It is the public city-layer entry point that connects people, places, opportunities, communities, events, and proofs into a clearer path.

OMDALAT exists so that local activity in Da Lat can become visible, trusted, and repeatable. The public site explains the value proposition. The docs layer on OMDALA explains the system rules and pathways. The app runtime on OMDALA manages activity and action.
