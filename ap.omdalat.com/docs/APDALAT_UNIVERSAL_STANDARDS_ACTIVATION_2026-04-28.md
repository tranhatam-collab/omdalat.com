2. Điều này làm thay đổi Ap Dalat ở đâu

2.1 Content và language

Mọi page, story, gallery, caption, CTA, meta, alt text phải:

* có tiếng Việt chuẩn
* có tiếng Anh chuẩn
* không trộn ngôn ngữ vô cớ
* không giống brochure du lịch
* không hype

2.2 SEO

Mọi route public của Ap Dalat phải có:

* title riêng theo ngôn ngữ
* meta description riêng theo ngôn ngữ
* canonical đúng
* hreflang đúng
* alt text ảnh riêng
* internal link đúng locale

2.3 Frontend

Mọi cải tiến UI phải bám:

* layout section rõ
* image-led design
* spacing và typography nhất quán
* mobile behavior ổn định
* accessibility cơ bản đầy đủ

2.4 Release

Không page nào của Ap Dalat được gọi là ready nếu chưa có:

* QA evidence
* route verification
* language verification
* sitemap/robots/meta verification
* rollback note nếu chạm release
