## 2. Applied Gate 0 Decision

Applied decision: `Option B for public brand language`, with a controlled compatibility note for OMDALA only where system architecture still requires it.

Reason:
- The foundation lock is more human, clearer, and better aligned with the current brand promise: live, work, learn, build in Dalat.
- Public users should not have to understand internal global-system architecture before understanding the offer.
- Developer/docs teams can still keep internal architecture references where needed, but public copy should lead with `Ôm Đà Lạt`.
