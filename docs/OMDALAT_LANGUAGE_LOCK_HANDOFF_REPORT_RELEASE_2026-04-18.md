# OMDALAT LANGUAGE LOCK FOR HANDOFF, REPORT, EVIDENCE, AND RELEASE

Phiên bản: 1.0.0  
Trạng thái: LOCKED  
Ngày cập nhật: 2026-04-18  
Chủ sở hữu: Team 1

## 1. Mục đích

File này khóa chuẩn ngôn ngữ cho toàn bộ:

- file giao việc
- file báo cáo
- evidence packet
- release gate
- release notes

Từ thời điểm này, bất kỳ file nào thuộc các nhóm trên mà:

- tiếng Việt sai dấu
- tiếng Anh sai nghĩa kỹ thuật
- dùng sai vai trò ngôn ngữ

thì đều **không đạt chuẩn review**.

## 2. Nguyên tắc ngôn ngữ bắt buộc

### Tiếng Việt

- Phải có dấu đầy đủ.
- Phải rõ nghĩa, ngắn gọn, đúng vai trò vận hành.
- Không viết kiểu ASCII không dấu cho tài liệu chính thức.
- Không dùng giọng quảng cáo, hô hào, hoặc mơ hồ.

### Tiếng Anh

- Chỉ dùng khi thật sự cần cho nghĩa kỹ thuật, tên chuẩn, hoặc nhãn song ngữ.
- Phải chính xác, nhất quán với codex hiện hành của Ôm Đà Lạt.
- Không dịch từng chữ nếu làm sai nghĩa.
- Không dùng startup hype, marketing hype, hoặc từ phóng đại.

## 3. Vai trò ngôn ngữ

- Tiếng Việt là ngôn ngữ gốc của file nội bộ và file phát hành cho team.
- Tiếng Anh là ngôn ngữ phụ trợ kỹ thuật hoặc bản đối chiếu chính xác.
- Không trộn hai ngôn ngữ trong cùng một câu nếu không thật sự cần.

## 5. Những gì không được dùng

Không dùng trong file chính thức:

- tiếng Việt không dấu
- câu nửa Việt nửa Anh vô cớ
- từ ngữ kiểu “premium”, “VIP”, “exclusive”
- cụm marketing như “game-changing”, “revolutionary”, “life-changing”

## 6. Bộ file chính thức đang dùng

Từ thời điểm này, Team 1 dùng các file sau làm nguồn chính thức:

- `RELEASE_CHECKLIST.md`
- `docs/OMDALAT_LIVE_ROUND2_TEAM_STATUS_2026-04-17.md`
- `docs/OMDALAT_TEAM1_ACTIVE_HANDOFF_2026-04-18.md`
- `docs/OMDALAT_EVIDENCE_PACKET_ROUND2_2026-04-18.md`
- `docs/OMDALAT_RELEASE_NOTES_ROUND2_2026-04-18.md`

Các file cũ không dấu chỉ được xem là tài liệu lịch sử hoặc bản nháp cũ, không dùng làm mốc review chính thức.

## 7. Tiêu chí pass review ngôn ngữ

Một file chỉ được xem là pass khi:

- tiếng Việt có dấu đầy đủ
- tiếng Anh đúng nghĩa kỹ thuật
- thuật ngữ nhất quán
- không có lỗi vai trò ngôn ngữ
- không có câu mơ hồ hoặc giọng điệu lệch codex
