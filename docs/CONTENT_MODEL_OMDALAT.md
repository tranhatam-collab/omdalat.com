# CONTENT_MODEL_OMDALAT
## Version 1.0

---

# 1. Principles

All content in OMDALAT must be:

- real
- local
- specific
- structured
- linkable

Avoid:

- generic filler
- tourism copy
- vague hype
- duplicate intros

---

# 2. Place Content Model

Required fields:

- name
- slug
- place type
- area
- short summary
- full description
- activity mode
- trust label
- verification state
- related hosts
- related communities
- related events
- related proofs
