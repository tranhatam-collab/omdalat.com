# LILY IMAGE REALITY STANDARD 2026

## 1. Principle

Lily’s visual system must show the real property, real rooms, real garden, real workspace and real activities. Images are evidence-bearing assets, not generic decoration.

## 2. Required shot list

### Property and access
Entrance, exterior, road/access context, signage when available, daytime and relevant safety context.

### Every room
Four corners, bed, storage, window, workspace, bathroom relationship, power points and any limitation.

### Shared spaces
Kitchen, common area, workspace, café corner, workshop area and circulation routes.

### Garden
Wide context, paths, active areas, tools/storage, hazards and seasonal state.

### Operations
Check-in setup, workspace use, learning/work session and garden activity only with participant consent.

## 3. Asset record

Every asset stores:

- asset ID;
- capture date;
- exact room/space reference;
- photographer/rightsholder;
- source type;
- original file hash;
- consent status for people;
- licence and permitted channels;
- alt VI/EN;
- caption VI/EN;
- crop variants;
- expiry/review date;
- revocation status.

## 4. Forbidden

- stock image represented as Lily;
- AI-generated room/garden represented as reality;
- images from Google, Booking, Facebook or guests without rights;
- misleading wide-angle edits;
- removal of material defects;
- image of a child without guardian consent and safeguarding review;
- passport, screen, address detail or private document visible accidentally;
- duplicated room image used for multiple rooms.

## 5. AI image use

AI-generated visuals may be used only for clearly labelled concept diagrams, never to represent the real accommodation. They must not appear in room inventory, location, gallery, maps, safety or availability contexts.

## 6. Public image gate

```text
real source
+ room/space match
+ rights verified
+ people consent verified
+ privacy checked
+ alt/caption approved
+ no misleading edit
= publishable
```

## 7. Revalidation

Room and key-space images are reviewed when:

- furniture or capacity changes;
- material damage/renovation occurs;
- a resident disputes accuracy;
- rights are revoked;
- twelve months pass without review.
